# Part 1 - Semantic Versioning

## Git Tag Documentation

### How to see tags 

```
git tag
```
### Make tag current commit using semantic versioning

```
git tag -a vx.x.x -m "message for tag"
```

### Push Tag to GitHub

```
git push origin (name of tag)
```
